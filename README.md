# Sistema de Gestão

Sistema web desenvolvido em Python utilizando Flask para gerenciamento de clientes, produtos, estoque e CRM.

## 🚀 Funcionalidades

- Login de usuário
- Dashboard com indicadores
- Cadastro de clientes
- Cadastro de produtos
- Controle de estoque
- CRM de leads
- Interface web responsiva

## 🛠️ Tecnologias utilizadas

- Python 3
- Flask
- Flask-SQLAlchemy
- SQLite
- HTML5
- CSS3
- Bootstrap

## 📦 Instalação

Clone o repositório:

bash
git clone https://github.com/arrozsemdoce/sistema_gestao.git
